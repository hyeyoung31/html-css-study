자손 선택자 - Descendant Selector.

자손 선택자는 문서 구조에서 앞서 선택한 선택자의 자손을 선택합니다.

문법

        문법: A B {속성: 값;}

자식 선택자 - Child Selector

문법

        A > B {속성: 값;}

인접 형제 선택자 - Adjacent sibilings Selector

앞선택자의 형제요소 중 앞 선택자 바로 다음에 따라오는 요소를 선택.

문법

        A + B {속성:값;}

일반 형제 선택자 - General siblings Selector

앞 선택자 뒤에 따라오는 형제 요소들 모두를 선택

문법

        A ~ B {속성:값;}

속성 선택자 - Attribute Selector

속성 선택자는 HTML 태그의 속성들을 통해 원하는 요소를 선택할 때 사용한다

문법

         [속성명] {속성:값;}
         [속성명= "값"] {속성:값;}
         [속성명^="값"] {속성:값;}

의사 클래스 선택자- Pseudo-class Selector

        :hover    요소에 마우스를 올렸을 때
        :link     a 태그 모두 선택
        :visited  이미 방문한 링크 선택
        :active   a,input,button 등의 요소를 누르는 순간
        :focus    탭 키를 통해 a, input, button 등의 요소에 포커스가 적용되었을 때
        :focus-within  자식요소 중 포커스가 적용되었을 때

의사 요소 선택자- Pseudo-element Selector

의사 요소 선택자는 가상으로 만들어진 요소를 선택하는 것이다.

         ::before, ::after

일반 구조 선택자 -Pseudo-class Selector

일반 구조 선택자는 형제 요소에서 인덱스번호를 이용하여 요소를 선택한다

         :first-child   선택자 중 첫 번째 요소를 선택
         :last-child    선택자 중 마지막 요소를 선택
         :nth-child(수열)  선택자 중 수열 번째의 요소를 선택
         :nth-last-child(수열)  선택자 중 마지막에서 수열 번째의 요소를 선택

짝수,홀수번째

         :nth-child(2n), :nth-child(even)
         :nth-child(2n+1), :nth-child(odd)

이상,이하

:nth-child(n+3) 3번째 이상의 모든 요소를 선택 :nth-child(-n+3) 3번째 이하의 모든 요소를 선택 :nth-child(n+3):nth-child(-n+6) 3번째 이상 6번째 이하의 요소를 선택

형태 구조 선택자 -Pseudo-class Selector

형태 구조 선택자는 같은 종류들 중 인덱스 번호를 이용하여 요소를 선택한다

        :first-of-type: 형제요소에서 같은 종류 중 첫번째 요소
        :last-of-type : 형제요소에서 같은 종류 중 마지막 요소
        :nth-of-type(수열) : 형제요소에서 같은 종류 중 수열 번째 요소

nth-child와 비교

        h2:first-child : 첫 번째 자식요소 중 h2태그를 찾음
        h2:first-of-type : h2 태그 중 첫번 째 요소

상태 선택자 -Pseudo-class Selector

상태 선택자는 주로 input, button 태그에 많이 사용한다

        input:disabled 사용불가 상태인 input 요소를 선택
        input:checked  체크된 input 요소를 선택
        :traget        클릭된 링크의 타켓을 받은 요소를 선택

nth-child와 비교

        h2:first-child : 첫 번째 자식요소 중 h2태그를 찾음
        h2:first-of-type : h2 태그 중 첫번 째 요소

부정 선택자 -Pseudo-class Selector

:not으로 선택하고 괄호 안에 요소의 반대 요소를 선택한다

        :not(선택자)
