text-indent

    ·글자에 들여쓰기 또는 내어쓰기를 적용
    ·양수는 들여쓰기
    ·음수는 내어쓰기

white-space

    ·글자의 줄 넘기 속성을 적용
    ·기본값은 normal로 다음 줄로 줄 넘김 적용
    ·pre는 space bar와 enter로 생성한 공백을 보존
    ·white-space:nowrap; white-space:normal; white-space:pre;

text-transform

    ·uppercase:대문자로
    ·lowercase:소문자로
    ·capitalize:첫글자 대문자로

vertical-allign

    ·글자의 수직 정렬
    ·인라인 요소들이 연이어 나열될 때 뒤따라오는 인라인 요소의 높이를 조절할 때 주로 사용
    .Baseline(기본값), sub, super, top, text-top, middle, bottom, text-bottom
    .이미지 외부에 테두리가 있을 때 vertical-allign:top; 을 쓰면 이미지 밑 공백이 없어짐. 
    .인라인 블록 요소인 비디오에서도 위에 동일하게 사용

white-space

    ·문단에서 너비가 한정되어 있을 때 내부의 글자를 아래로 넘겨서 쌓을지 여부를 결정
    ·nowrap : 아래로 줄 넘김하지 않는다
    .normal: 아래로 줄 넘김한다.
    .pre: 글자 사이의 공백과 엔터를 그대로 보존한다. 끈 부분에서 넘치는 텍스트는 아래로 줄 넘김을 한다.
    .pre-wrap : 공백과 enter유지
    .pre-line : 공백은 유지x , enter(줄바꿈)유지

line-height

.인라인 요소일 때 적용이 안되고 블록 요소일 때 반영
