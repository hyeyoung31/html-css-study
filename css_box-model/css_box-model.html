Box Model

     ①박스 모델의 개념
     content를 중심으로 padding, border, margin영역으로 구성
     ·영역별 설명:
     content: 실제 내용이 표시되는 영역으로 텍스트와 이미지가 위치
     padding: content 주변의 여백으로 기본적으로 투명
     border: padding 바깥 영역의 테두리를 표시
     *배경은 border 영역까지 나옴
     magin: border 바깥 영역의 여백을 표시하고 투명
     ②content-box
     ·기본값으로서 요소의 너비를 지정했을 때 content 영역의 크기를 지정
     ·width:500px;
     
     ③border-box
     ·너비를 지정하면 border 영역의 크기를 설정
     · width:500px;
     · 500px = content + padding +border
     · 배경을 넣었다면 border 가 있다면 border까지, padding 이 있다면 padding까지 , content만 있다면 content까지
속성 값 설명
Padding Border Margin 10px 모든 방향 10px
10px 20px 상하10px, 좌우 20px
10px 20px 5px 상10px, 좌우 20px, 하5px
10px 20px 5px 1px 상10px, 우20px, 하5px, 좌1px
