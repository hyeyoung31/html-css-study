Box model 박스 모델

  ·content: 실제 내용이 표시되는 영역으로 텍스트와 이미지가 위치
  ·padding: content 주변의 여백으로 기본적으로 투명
  ·border: padding 바깥 영역의 테두리를 말함
  ·margin: border 바깥 영역의 여백을 표시하고 투명함

box-sizing

    ·box-sizing: content-box  width값은 content의 크기를 지정한 것
    ·box-sizing: border-box   width값은 padding,border까지의 크기를 지정한 것

padding, border, margin


    · 10px : 상후하좌 모든 방향 10px
    · 10px 20px : 상하 10px, 좌우 20px
    ·10px 20px 5px : 상 10px, 좌우 20px, 하5px
    ·10px 20px 5px 1px : 상 10 px, 우 20px, 하 5px, 좌 1px
    ·padding-right, padding-left, padding-top, padding-bottom 같이 한쪽만 지정할 수 있음
    

    
border상하좌우 스타일 다르게 주기

    
    .box{
     width:500px;
     padding:20px;
     borer-width: 10px 20px 5px;
     border-style: solid dashed dotted;
     border-color ; #ccc #000 #f00;
    }
    


    
margin auto

    
    ·margin:0 auto; 좌우 여백을 자동으로 배치하여 요소를 가로측에서 화면 가운데 배치할 수 있다.
    ·margin:0 auto는 margin:auto 와 같이 줄여서 표현할 수 있다.
