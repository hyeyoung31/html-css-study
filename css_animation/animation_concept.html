animation은 페이지 열리자마자 사용자 이벤트와 상관없이 작동하는 애니메이션

animation-name 

    ·animation-name: keyframe name | none(기존 작성없앨 때) | initial |inherit;
    ·keyframse
     구간별 변경될 CSS 속성을 기술
     구간 설정은 %(퍼센트) 또는 from,to로 지정할 수 있음
     ex) 0% {속성:값} 100%{속성:값}

animation-duration 

        ·애니메이션 하나의 사이클이 끝날 때까지의 시간을 지정
        ·animation-duration : time | initial | inherit;
        ·time: 필수값, 값이 없으면 기본값 0, 초(s), 밀리초(ms)

animation-delay 

        ·애니메이션 시작 전 지연시간을 설정
        ·animation-delay : time | initial | inherit;
        ·time: 옵션값, 입력하지 않으면 기본값 0, 초(s),밀리초(ms)
        음수를 지정하면 지정한 시간 만큼 경과한 위치에서 시작

animation-iteration-count 

        ·애니메이션의 반복 횟수를 지정
        ·animation-iteration-count: number |infinite | initial | inherit;
        ·number: 기본값1, 애니메이션의 재생 회수
         infinite: 무한반복

animation-timing-function 

        ·css속성의 값이 변화되는 시간을 정의하여 속도를 설정
        ·animation-timing-function:linear | ease | ease-in | ease-out | ease-in-out |
         step-start | step-end | steps(int, start|end)| cubic-bezier (n,n,n,n) |initial |inherit;
        ·number: 기본값1, 애니메이션의 재생 회수
         infinite: 무한반복

animation-direction 

        ·애니메이션의 진행 방향을 순방향, 역방향 등으로 지정
        ·normal : 기본값, 순방향 즉, @key frames 에서 0% ->100% 순서
        ·reverse : 역방향으로 @keyframes에서 100% -> 0% 순서
        ·alternate : 순방향에서 역방향으로 @keyframes에서 0% -> 100% -> 0%
        ·alternate-reverse : 역방향에서 순방향으로 @keyframes 에서 100% -> 0% ->100%

animation-fill-mode 

        ·CSS 애니메이션이 실행 전과 후에 대상에 스타일을 적용하는 방법을 지정
        ·animation-fill-mode: none | forwards |backwards | both
        ·none: 기본값으로 animation 시작하 전에 스타일에 어떤 영향도 주지 않음
        ·forwards : 애니메이션이 끝나면 keyframe의 마지막 스타일을 유지
        (animation-direction에 따라 달라질 수 있음)
        ·backwards: 애니메이션이 끝나면 keyframe의 첫 번째 스타일을 유지
        .both : 앞, 뒤 양쪽 모두의 규칙을 따르므로 애니메이션 속성이 양방향으로 확장
